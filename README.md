# synthizer_cs
A binding to the [synthizer](https://github.com/synthizer/synthizer) Audio library.
# What currently works.
- Context.
- Buffer
    - FromEncodedData
    - FromFile
- Generator
    - StreamingGenerator
        - FromFile
        - FromStreamHandle
    - BufferGenerator
- Source
    - DirectSource
    - Source3D
- Properties, Almost all.
- StreamHandle
    - FromMemory
    - FromFile
